Reign: A Game of Lords and Leaders (Foundry VTT)
An unofficial, fan-made system for playing the Reign tabletop RPG on Foundry VTT, powered by the One Roll Engine (ORE).

Features
Automated ORE dice rolling (Sets, Waste, Expert/Master Dice)

Combat automation (Hit locations, AR mitigation, area splash)

Company Warfare & Minion Horde (Threat) mechanics

XP progression engine
